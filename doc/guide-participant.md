# Guide Participant — Pixel Night

Bienvenue ! Ce guide explique comment participer à une soirée Pixel Night.
**Aucun compte n'est nécessaire.**

## Rejoindre une soirée

1. Ouvrir l'application Pixel Night (lien fourni par votre organisateur)
2. Sur la page d'accueil, saisir le **code de la salle** (exemple : `cine-des-potes`)
3. Cliquer **Rejoindre**

Vous pouvez aussi accéder directement via un lien comme `/s/cine-des-potes`.

## Voter pour un thème

Quand la soirée est en phase de vote thème :

1. Vous voyez les thèmes proposés (ex : Western, Science-fiction, Comédie...)
2. Cliquez sur le thème que vous préférez
3. Votre vote est enregistré instantanément — vous ne pouvez voter qu'une seule fois

> Une fois voté, le bouton est désactivé. Vous verrez la confirmation de votre vote.

## Proposer un film (si activé)

Si l'organisateur a activé la phase de propositions, vous pouvez suggérer des films avant
le vote final.

1. Le thème gagnant est affiché
2. Utilisez le champ de recherche pour chercher un film sur TMDb
3. Cliquez **Proposer** sur le film que vous souhaitez soumettre
4. Vous pouvez proposer **jusqu'à 3 films**

> La phase de propositions a une durée limitée. Une fois clôturée, vous ne pouvez plus proposer.

## Voter pour un film

Quand la soirée est en phase de vote film :

1. Vous voyez les films sélectionnés, avec leur affiche, titre, durée et réalisateur
2. Cliquez sur le film que vous voulez voir
3. Votre vote est enregistré — vous ne pouvez voter qu'une seule fois

> En cliquant sur la bande-annonce (si disponible), vous pouvez voir un aperçu du film sur YouTube.

## Voir le résultat

Une fois le vote terminé, la soirée passe en mode "Terminée".
Le **film gagnant** est affiché pour tous les participants.

## Questions fréquentes

**Ai-je besoin d'un compte ?**
Non. Aucune inscription, aucun email requis.

**Comment mon vote est-il identifié ?**
Un identifiant anonyme (code unique) est créé dans votre navigateur la première fois.
Aucune donnée personnelle n't est collectée.

**Puis-je changer mon vote ?**
Non — une fois voté, c'est définitif.

**Puis-je voter depuis mon téléphone ?**
Oui, l'application fonctionne sur mobile.

**La soirée est en phase "Annulée" — que faire ?**
L'organisateur a annulé la soirée. Contactez-le pour plus d'informations.

**Je ne vois pas encore les films / thèmes — pourquoi ?**
La soirée est peut-être encore en phase d'attente (`planned`).
L'organisateur doit lancer le vote pour que les thèmes apparaissent.
Rechargez la page dans quelques instants.
